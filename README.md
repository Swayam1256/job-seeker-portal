💼 Job Seeker Portal (MERN Stack)

A full-stack Job Seeker & Employer Portal built using the MERN Stack where employers can post jobs and manage applications, and job seekers can browse jobs, apply, and manage their profiles.

🚀 Live Features
🔐 Authentication

User signup & login

JWT-based authentication

Role-based access (Employer / Seeker)

🧑‍💼 Employer Features

Post new job listings

View all posted jobs

View applications separately for each job

Accept or reject applicants

Delete jobs

👩‍💻 Job Seeker Features

Browse all available jobs

Search jobs by title and location

View detailed job description

Apply to jobs

Track application status

Manage profile & upload resume

🎨 UI / UX

Modern responsive UI with Tailwind CSS

Clean dashboards for Employer & Seeker

Toast notifications for actions

Beautiful job cards & detail pages

🛠️ Tech Stack
Frontend

React.js

Vite

Tailwind CSS

React Router

Axios

Lucide Icons

React Hot Toast

Backend

Node.js

Express.js

MongoDB (Mongoose)
